# Homebrew Tap for ralphtools

## Installation

```bash
brew tap EtanHey/ralphtools
brew install ralphtools
```

## Usage

```bash
ralph --help
ralph 50          # Run 50 iterations
ralph config      # Configure settings
```

## Upgrade

```bash
brew upgrade ralphtools
```

## Uninstall

```bash
brew uninstall ralphtools
brew untap EtanHey/ralphtools
```

## What is ralphtools?

An autonomous coding loop that executes PRD stories using AI models (Claude, Gemini, Kiro). It spawns fresh AI instances in a loop, each working on one story until all are complete.

Learn more: https://github.com/EtanHey/ralphtools
